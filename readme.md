# Ascient

A backend for storing generic data.

## Supported Datatypes

* Booleans

## Planned Datatypes

* Lists
  * Queue manipulation methods
  * Stack manipulation methods
* Numbers
  * Counters
  * Arbitrary math statements
  * Stored math events
  
## Other planned stuff

* Last modified date for all datatypes
* Creation date for all datatypes
* Triggers for booleans
  * Timers
  * Dates
